Logic

Logical expressions can contain logical operators such as: AND $(\land)$, OR $(\lor)$, and NOT $(\lnot)$. When the values of the operands of the logical operators in a logical expression are known, the value of the expression can be determined using rules such as:

  1. The expression $p\land q$ is true only when both $p$ and $q$ are true; it is false otherwise.
  2. The expression $p\lor q$ is true if either $p$ or $q$, or both are true; it is false otherwise.
  3. The expression $\lnot p$ is true if $p$ is false, and false if $p$ is true.

Exclusive OR

It gains the name "exclusive or" because the meaning of "or" is ambiguous when both operands are true; the exclusive or operator excludes that case. This is sometimes thought of as "one or the other but not both". This could be written as "A or B, but not, A and B".

The XOR ($\oplus$) logical operator outputs true whenever either p or q is true, but not both; it is false otherwise.

Logical equality

Logical equality is a logical operator that corresponds to equality in Boolean algebra It gives the functional value true if both functional arguments have the same logical value, and false if they are different. It is also logically equivalent to $(p \land q) \lor (\lnot p \land \lnot q)$ ot the XNOR (exclusive not or) boolean operator, meaning "both or neither".

The logical equality ($\leftrightarrow$) logical operator outputs true whenever both p and q are true or both are false; it is false otherwise.
