Cartesian Products

Let $A$ and $B$ be two sets. Then theproduct of $A$ and $B$, denoted $A \times B$, isdefined as the set of pairs in which the component is chosen from $A$ and thesecond component from $B$. That is $A \times B = \{(a, b)\mid a \in A \land b \in B\}$.

NOTE: The product of a set A with the empty set is the empty set i.e. $A \times \emptyset = \emptyset$
$$A = \emptyset$$
$$B = \emptyset$$
$$A \times B = \emptyset$$
